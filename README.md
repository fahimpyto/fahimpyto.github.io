# fahimpyto.github.io

Personal portfolio of **Fahim Ahmed** — Python Automation Expert, Technical SEO Practitioner, Open Source Developer, and AI Enthusiast.

Built with [Astro](https://astro.build) and [Tailwind CSS](https://tailwindcss.com).

## Tech Stack

- **Framework:** Astro 7
- **Styling:** Tailwind CSS 4
- **Content:** MDX
- **Fonts:** Inter, JetBrains Mono, Space Grotesk
- **Icons:** Lucide
- **Deployment:** GitHub Pages

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```
